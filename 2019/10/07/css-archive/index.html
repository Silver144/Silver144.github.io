background

设置图片:

background-image: url('url');

不重复:

background-repeat: no-repeat;

水平平铺:

background-repeat: repeat-x;

图片位置右上

background-position: left bottom;

简写按顺序分别是color url repeat attachment(fixed表示图片不动, 否则scroll)

Text

对齐

text-align: center(居中) right(右对齐) justify(自适应)

线条

text-decoration: overline line-through underline(默认)

变换

text-transform: uppercase lowercase capitalize(首字母大写)

缩进

text-indident: n px

字符之间空间

letter-spacing: n px

行间

line-height: 100%

Font

字体系列

font-family: a, b, "c d"

字体大小

font-size: 1em 100%

Link

link.link: { background-color: #FFFFFF; }

link.visited: { background-color: #FFFFFF; }

link.hover: { background-color: #FFFFFF; }

link.active: { background-color: #FFFFFF; }

List

无序列表

ul.example { list-style-type: circle square; }

有序列表

ul.example { list-style-type: upper-roman upper-alpha digit(默认); }

Table

边框大小

设置折叠

table { table-collapse: collapse; }
